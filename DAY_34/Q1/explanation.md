# Day 34 - Question 1 Solution

## 🎯 Problem
Evaluate Postfix Expression

## 💡 Approach
The program uses a linked-list based stack to store operands. When an operator is encountered, it pops the top two elements, performs the operation, and pushes the result back.

## ⏱️ Time Complexity
O(n)

## 💾 Space Complexity
O(n)
