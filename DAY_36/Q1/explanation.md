# Day 36 - Question 1 Solution

## 🎯 Problem
Circular Queue Using Array

## 💡 Approach
Use a fixed array as a circular buffer; enqueue all inputs, then advance the front pointer m times to simulate dequeues, finally print from front to rear in order.

## ⏱️ Time Complexity
O(n+m)

## 💾 Space Complexity
O(1)
