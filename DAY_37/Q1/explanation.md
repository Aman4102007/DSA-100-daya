# Day 37 - Question 1 Solution

## 🎯 Problem
Priority Queue Using Array

## 💡 Approach
Store elements in an array; for delete and peek scan the array to find the minimum element, output it, and for delete shift remaining elements left.

## ⏱️ Time Complexity
O(n) per delete/peek, O(1) for insert

## 💾 Space Complexity
O(1)
