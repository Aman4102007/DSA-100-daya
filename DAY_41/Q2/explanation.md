# Day 41 - Question 2 Solution

## 🎯 Problem
Task Scheduler

## 💡 Approach
Count task frequencies, find the most frequent count and how many tasks share it; compute idle slots using (max-1)*(n+1)+cnt and return the max of that and total tasks.

## ⏱️ Time Complexity
O(tasksSize + 26)

## 💾 Space Complexity
O(26)
