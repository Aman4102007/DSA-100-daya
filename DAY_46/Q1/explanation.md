# Day 46 - Question 1 Solution

## 🎯 Problem
Level Order Traversal

## 💡 Approach
Read the number of nodes and the node values stored in an array; print them in the same order, which is the level‑order traversal for a complete binary tree represented by an array.

## ⏱️ Time Complexity
O(n)

## 💾 Space Complexity
O(n)
