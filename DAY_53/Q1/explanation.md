# Day 53 - Question 1 Solution

## 🎯 Problem
Print Binary Tree Vertical Order

## 💡 Approach
Build the tree from level‑order input, compute horizontal distance for each node with BFS, then print nodes grouped by distance from leftmost to rightmost.

## ⏱️ Time Complexity
O(n)

## 💾 Space Complexity
O(n)
