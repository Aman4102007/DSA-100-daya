# Day 82 - Question 1 Solution

## 🎯 Problem
Upper Bound & Lower Bound Implementation

## 💡 Approach
Binary search is used twice: once to find the first index with value >= x (lower bound) and once for the first index with value > x (upper bound).

## ⏱️ Time Complexity
O(log n)

## 💾 Space Complexity
O(1)
